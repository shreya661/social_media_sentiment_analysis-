### Social_Media_Sentiment_Analysis
 #1.Description
• This project focuses on analyzing social media data to understand how people feel about different posts.  
• The main goal is to identify whether the sentiment of posts is positive or negative and to present the results using simple and interactive visualizations.
•I  built this project to improve my understanding of Python, data analysis, and data visualization concepts at a beginner level.

#3. Installation Instructions
Follow these steps to run the project locally:
* Clone the repository:
   ```bash
   git clone https://github.com/shreya661/sentiment-analysis-project.git
* Open the project folder.
* Install the required libraries:
         pip install pandas plotly
* Open the Jupyter Notebook and run the cells.
 Prerequisites:
       • Python (basic knowledge)
       • Jupyter Notebook
       • Pandas and Plotly libraries
#4.Usage
• Run the Jupyter Notebook to load and clean the dataset
• Analyze sentiment data based on platforms and countries
• Generate visualizations such as bar charts and pie charts
• View interactive graphs in the browser or exported
#5.Features
• Beginner-level sentiment analysis
• Platform-wise post comparison
• Interactive visualizations using Plotly
• Easy-to-understand workflow for learning purposes
#6.Data Sources
The dataset used in this project contains:
         • Social media platform names
         • Country information
         • Sentiment labels (Positive / Negative)
         • The dataset is used only for practice and learning purposes.
         • Hashtags and post content
#7. Methodology
• Data cleaning and preprocessing using Pandas
• Grouping and counting sentiment values
• Visualizing insights using bar and pie charts
• No advanced machine learning techniques are used, making this project suitable for beginners.       
#8. Insights and Results
• Identified overall sentiment trends across platforms
• Observed differences in user engagement between platforms
• Visualized how users react to different types of social media content
